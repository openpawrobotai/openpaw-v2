# Firmware

## Overview

This directory contains the firmware running on the ESP32-based BallBot controller.

The firmware is responsible for:

- Sensor acquisition
- Motor control
- Balance stabilization
- Communication interfaces
- OTA updates
- Safety systems

## Core Modules

### Sensors

- IMU
- Encoders
- Distance Sensors

### Control

- PID Controller
- Motion Control
- State Management

### Connectivity

- WiFi
- Bluetooth
- API Server

## Development Goals

- Stable balancing
- Low latency control
- Modular architecture
- Easy debugging
