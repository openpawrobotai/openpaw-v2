# 🎯 Project Milestones

## Overview

This document tracks the major milestones for the OpenPaw BallBot project.

Each milestone represents a significant step toward building a fully functional self-balancing robotic platform powered by ESP32.

---

# Milestone 1 — Research & Planning

## Goal

Establish the technical foundation of the project.

### Deliverables

* Project requirements defined
* Initial research completed
* Architecture documents created
* Technology stack selected
* Repository structure established

### Status

🟢 Completed

# Milestone 5 — Self-Balancing Control

## Goal

Achieve stable balancing on the ball platform.

### Deliverables

* IMU calibration
* State estimation
* PID controller
* Balance stabilization
* Disturbance recovery

### Success Criteria

✅ Robot balances continuously

✅ Robot recovers from small pushes

✅ Stable operation for extended periods

### Status

⚪ Planned
