# Hardware Design

## Overview

This directory contains hardware documentation for OpenPaw BallBot.

The hardware platform includes:

- ESP32 Controller
- IMU Sensor
- Motor Drivers
- BLDC Motors
- Power Distribution System
- Connectivity Modules

## Documentation

Future documents may include:

- Block diagrams
- Wiring diagrams
- Component selection
- Assembly instructions
- BOM (Bill of Materials)

- Reliable operation

- Low cost
- Open-source design
