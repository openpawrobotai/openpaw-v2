# 📓 OpenPaw Devlog

A public, day-by-day diary of building OpenPaw. **One entry per working day.**
This doubles as our build-in-public content and our memory of what/why.

## How to add today's entry
1. Copy `TEMPLATE.md` → `YYYY-MM-DD.md` (e.g. `2026-06-02.md`).
2. Fill it in (2 minutes). **Always include a demo** — a GIF/screenshot in `media/`
   or a link to the Discord clip / YouTube video.
3. Commit & push:
   ```bash
   git add devlog/ && git commit -m "[post] devlog: <one-line summary>" && git push
   ```
   The `[post]` marker turns it into a Discord/Telegram line + an X draft.

## Rules
- Short and honest beats long and polished. Show broken things too.
- No secrets, no private `openpaw-app`/AI details.
- Put images in `devlog/media/` (keep them small — GIFs < 5 MB; big videos go to
  YouTube or the shared "OpenPaw Demos" Drive folder and get linked here).

See the full routine in [`../intern-guide.md`](../intern-guide.md).
